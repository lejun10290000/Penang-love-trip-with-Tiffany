Penang 4D3N Itemized Travel Budget Calculator

WHAT IS NEW
- The current draft saves automatically in the browser.
- Tap “Save history record” to store a separate snapshot.
- Open, share, or delete saved records.
- Export a JSON backup and import it on another phone.
- When hosted online, “Share current trip” creates a link containing a copy of the current trip.

IPHONE SETUP
1. For testing, extract the ZIP in Files and open index.html. Local-file browser storage may not be reliable.
2. For reliable use, upload every file in this folder to a static web host such as GitHub Pages. Keep index.html at the repository root.
3. Open the published link in Safari.
4. Tap Safari Share, then Add to Home Screen.
5. Send the same published link to the other traveller.

SHARING DATA
- Browser history is separate on each phone.
- Use Share current trip to send a copy.
- Or export a backup file and let the other person import it.
- Live two-way synchronization requires a cloud database and is not included in this offline/static version.
